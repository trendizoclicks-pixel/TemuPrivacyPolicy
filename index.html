3. How We Use Information

All data processing is performed locally in your browser:

  • Data Extraction: Parse Temu.com HTML to extract product information
  • Data Formatting: Structure data for CSV, Excel, or JSON export
  • Local Storage: Save preferences and extracted data between sessions
  • UI Display: Show extracted data in the extension popup

5. Data Storage & Processing

Local Processing Only: All data extraction, processing, and storage happens within your Chrome browser. No data leaves your computer.

Storage Location: Data is stored using Chrome's chrome.storage.local API. This data:

  • Persists until you clear it or uninstall the extension
  • Is not synced to Google servers
  • Is not accessible by other extensions or websites
  • Can be cleared anytime via the "Clear Data" button

6. Third-Party Libraries

We use the following open-source libraries:

  • SheetJS (xlsx.js): For Excel file generation. Runs locally in your browser.

7. Your Rights & Control

You have complete control over your data:

  • Clear Data: Use the "Clear Data" button to remove all stored information
  • Choose Data: Select which product fields to extract and export
  • Uninstall: Removing the extension deletes all locally stored data
  • Export Data: Download your data in CSV, Excel, or JSON format

8. Children's Privacy

Our extension is not directed to individuals under the age of 13. We do not knowingly collect personal information from children under 13.
